# Google Play Store Data Analysis (Python)

## Overview
This project performs exploratory data analysis (EDA) on Google Play Store app data to identify trends in app categories, ratings, and user engagement.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Analysis
- Data cleaning and preprocessing
- Category-wise app distribution
- Rating analysis
- Review-based popularity analysis

## Key Insights
- Apps with higher reviews tend to be more popular
- Categories like Games and Tools dominate the Play Store
- Most apps are free, with fewer paid apps
- Ratings are relatively consistent across categories

## Features
- Data cleaning and transformation
- Exploratory Data Analysis (EDA)
- Visualization using bar charts
